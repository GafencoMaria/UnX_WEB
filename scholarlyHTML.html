1. Introduction

1.1 Purpose

This document is meant to highlight the software requirements for the Web Technologies course project entitled UnX

1.2 Product Scope

UnX is a management software product that aims to provide its users with data regarding the statistics of the unemployment rate.

2. Overall Description

2.1 Product Functions

The web application is expected to perform the following functionalities:

  • Provide access to information regarding unemployment rate from different romanian cities.
  • Visualize the data using a chart bar, a bubble chart, and a map of the romanian cities.
  • The option to download the charts from the website in SVG/WebP format.
  • The option to download the available data in CVS format.

2.2 User Classes and Characteristics

There are two categories of users:

  1. Guest users have the following characteristics:
    • Access to overview statistics.
    • Can download the informations.
    • The users can compare the data.
  2. Admins
    • The application also provides access to a ranking of cities in terms of unemployment rate.
    • Additionally, a user can compare the information between different cities regarding joblessness.
    • The administrator of the application could access the database and could delete/modify the existing information.

2.3 Operating Environment

The following technologies have been used. It is important to mention that no frameworks have been used, according to the Web Technologies requirements.

Front-end technologies: HTML, CSS, JavaScript

2.4 Usage Manual

  1. Overview

    Upon accessing the platform, the user will see information regarding unemployment statistics in Romania and in the top part the navigation bar with the purpose of navigating through the pages.

  2. Criterii

    The dropdown that allows the users to pick and choose the criteria for the visualisation tool.

  3. Acasa

    The home page of the application.

  4. Despre noi

    General information regarding the creators of the web tool.

  5. Mai multe resurse

    More resources related to unemployment, as well as tips for job interviews.

  6. Login

    A login page exclusively for registered admins.
